### Classes - Classes provide a means of bundling data and functionality together. - Creating a new class creates a new type of object. ```python class AClass: x = 5 ```
### The \_\_init\_\_ Method - `__init__` is a special method to initialize an object out of a class by its attributes. ```python class Person: def __init__(self, name, age): self.name = name self.age = age ```
### Methods - Functions of a class are called methods used to define behaviors of an object. ```python class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): print(f"Hello, my name is {self.name}") ```
### Inheritance - Helps to define a class that inherits all the methods and properties from *another* class. ```python class Student(Person): # `Student` inheriting from `Person` def __init__(self, name, age, year): super().__init__(name, age) self.graduationyear = year ```
### Encapsulation - Encapsulation involves restricting access to methods and variables. - This prevents data from being modified directly, ensuring data integrity. ```python class Computer: def __init__(self): self.__maxprice = 900 def get_selling_price(self): print(f"Selling Price: {self.__maxprice}") ```
### Polymorphism - Polymorphism allows for methods in the child class to implement a method with the same name as defined in their parent class. - This supports different behaviors for the same method in different classes. ```python class Bird: ... def flight(self): print("Most of the birds can fly but some cannot.") class Sparrow(Bird): def flight(self): print("Sparrows can fly.") ```
### Class and Instance Variables - Class variables are shared by all instances of a class. - Instance variables are unique to each instance. ```python class Dog: animal = 'dog' # Class variable def __init__(self, name): self.name = name # Instance variable ```
