### Classes - Classes provide a means of bundling data and functionality together. - Creating a new class creates a new type of object. ```python class AClass: x = 5 ```
### The __init__ Method - `__init__` is a special method to initialize an object out of a class by its attributes. ```python class Person: def __init__(self, name, age): self.name = name self.age = age ```
